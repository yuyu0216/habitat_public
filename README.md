# 外來物種生態教育遊戲系統

NCU 114

**一款台灣在地、專為國中小學生設計的生態教育模擬遊戲**

讓玩家化身紅火蟻、綠鬣蜥、美國牛蛙、貢德氏赤蛙、蓋斑鬥魚、領角鴞，  
在**濕地農田、淺山森林、都會公園、港口物流**四種真實棲地中，  
透過**覓食、繁殖、遷移、探索**等行動，親身體驗：

> **入侵種如何爆發？原生種如何脆弱？人類活動又是如何一步步改變整個生態？**

---

### ✨ 遊戲特色

- **6 種台灣真實物種**：入侵種 vs 原生種的真實生態關係
- **4 種台灣代表性棲地**：每塊土地都有不同的資源與限制
- **多人協力機制**：每種物種由 5 位玩家共同操控，考驗團隊合作
- **探索卡系統**：透過目擊事件親眼「看到」掠食鏈與生態互動
- **人類事件系統**：農藥、道路開闢、林木砍伐、廢水… 讓玩家深刻體會「我們把環境搞壞了」
- **透明公式與平衡設計**：每一步都有跡可循，適合課堂實際運行

---

### 🎯 教育目標

透過 5 回合的遊戲，讓學生自然理解：
- 入侵種的擴散壓力與破壞力
- 原生種的生存困境與脆弱性
- 人類活動才是生態變化的最大推手
- 保護棲地、減少人為干擾的重要性

**玩完遊戲後，學生不再只是「知道」生態議題，而是真正「感受」過一次。**
